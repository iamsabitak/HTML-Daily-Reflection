Javascript

JavaScript is the world's most popular programming language. JavaScript is the programming language of the Web.

Why JavaScript?

Ecmascript is a standard for scripting languages, including JavaScript, JScript, and ActionScript. It is also best known as a JavaScript standard intended to ensure the interoperability of web pages across different web browsers. Javascript and Ecmascript can almost always be used interchangeably. Javascript is very liberal in what it allows. js can be executed right inside the browser.

About variables in Javascript

A variable in js means a container which contains a value. the value of a js variable can be changed during the execution of the program. var can be updated and redeclared with same variable let cannot be redeclared but can be updated. const can neither be updated nor be redeclared.
